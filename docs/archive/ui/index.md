# Archivo Historico UI

Este directorio guarda documentos del refactor visual cerrado en marzo de 2026.

## Que hay aqui

- auditoria visual previa,
- plan de ejecucion del refactor,
- resumen ejecutivo del trabajo completado.

## Para que sirve

- trazabilidad historica,
- contexto para decisiones visuales pasadas,
- soporte para revisar por que existe el sistema de tokens actual.

## Para que no sirve

- no reemplaza la documentacion activa,
- no define la arquitectura frontend vigente,
- y no deberia ser el primer lugar al que alguien vaya para cambiar una pantalla hoy.
